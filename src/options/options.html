Rule Capabilities

You can manage up to 5,000 custom rules. These rules persist across browser updates and restarts.

  • Block Everything: Completely blocks all connections to a specific domain.
  • Ad Server: Targeted blocking of scripts, frames, and objects used to deliver ads.
  • Analytics: Blocks invisible pixels, tracking scripts, and ping requests.
  • Regex Rules: Advanced pattern matching using RE2 syntax (Limit: 1,000 active regex rules).

Quick Actions (Popup)

The extension popup provides 3 global toggles that apply to all websites:

  • JS (Block JavaScript): Stops all script files from loading. Useful for extreme privacy or bypassing overlays.
  • IMG (Block Images): Saves bandwidth by preventing image loading.
  • VID (Block Videos): Stops all media streaming requests.
